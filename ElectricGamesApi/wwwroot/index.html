ElectricGames API

Api for games, character, and images

Endpoints

  • https://localhost:7063/game
  • https://localhost:7063/gameCharacter
  • https://localhost:7063/game/getByTitle
  • https://localhost:7063/gameCharacter/getByName
  • https://localhost:7063/image

Info about ElectricGames-Games

  • Id: int
  • Title: string
  • Image: string
  • Description: string
  • Platform: string
  • ReleaseYear: int

Info about ElectricGames-Characters

  • Id: int
  • Name: string
  • Image: string
  • Game: string
